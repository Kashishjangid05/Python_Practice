# 🐍 Python Practice & Projects Master Repository

Welcome to my **Python Practice** repository! 🚀 This repository serves as a comprehensive collection of my journey learning Python, starting from absolute basics to advanced concepts, alongside various problem-solving exercises and hands-on mini-projects.

---

## 📂 Repository Structure

This repository is intuitively organized into four main directories to keep everything modular and logically structured:

### 1️⃣ **Basics (`/Basics`)** 📖
Contains well-documented Python scripts focusing on the core building blocks of the language. If you want to review the syntax or basics, this is the place!
* **Concepts Covered**: Variables, Inputs, Operators, Data Structures (`Lists`, `Tuples`, `Dictionaries`, `Sets`), Control Flow (`ConditionalStatements`, `Loops`), Functions, Recursion, File I/O, and introductory Object-Oriented Programming (`Oops.py`).

### 2️⃣ **Practice Questions (`/PracticeQuestions`)** 🧠
A dedicated section for daily problem-solving logic. This folder contains a sequence of scripting challenges (`Practice1.py` through `Practice31.py`).
* **Goal**: To enhance algorithmic thinking, debug skills, and efficiently apply theoretical Python concepts to standard algorithmic problems.

### 3️⃣ **Training (`/Training`)** 🏋️‍♂️
A deeper, more rigorous dive into intermediate/advanced Python topics and real-life scenario scripts.
* **Topics Included**: Advanced Functions, Variable Scope (`GlobalLocalScope`, `Enclosing`), Exception Handling, Advanced Object-Oriented patterns (`constructorDestructor.py`), and complex Loop `Pattern.py` creation.
* **Highlights**: Also contains logical apps such as:
  * `BankApplication/`
  * `electricity_bill.py`
  * `Train_ticket.py`
  * `username_password.py`

### 4️⃣ **Projects (`/Project`)** 🛠️
Hands-on Python scripts showcasing the application of acquired skills into workable CLI logic.
* **Rent Calculator (`rentCalc`)**: A mathematical script to intelligently calculate and divide living expenses.
* **Rock Paper Scissors (`rockPaperScissor`)**: An interactive command-line implementation of the classic game against a randomized computer.

## 🚀 How to Run the Scripts

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Kashishjangid05/Python_Practice.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Python_Practice
   ```
3. **Execute any Python script manually from your terminal:**
   ```bash
   python Basics/FirstProgram.py
   # OR
   python Project/rockPaperScissor
   ```
